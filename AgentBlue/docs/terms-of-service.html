이용약관

최종 수정일: 2026년 2월 26일 · 시행일: 2026년 2월 26일

제1조 (목적)

이 약관은 AgentDroid(이하 "본 앱")가 제공하는 AI 기반 안드로이드 자동 조작 서비스의 이용 조건 및 절차, 이용자와 앱 운영자 간의 권리·의무 및 책임 사항을 규정함을 목적으로 합니다.

제2조 (서비스의 내용)

  1. 본 앱은 AI(인공지능) 모델을 활용하여 안드로이드 기기의 화면을 분석하고, 사용자의 자연어 명령에 따라 자동으로 화면 조작(클릭, 텍스트 입력, 스크롤 등)을 수행하는 에이전트 서비스를 제공합니다.
  2. 본 앱은 Android 접근성(Accessibility) API를 사용하여 화면 정보를 수집하고 조작을 수행합니다.
  3. 본 앱의 AI 분석 기능은 사용자가 직접 설정한 외부 AI API(OpenAI, Google Gemini, Anthropic Claude, DeepSeek)를 통해 제공되며, AI API 사용에 따른 비용은 사용자가 부담합니다.

제3조 (이용 조건)

1. 본 앱을 사용하기 위해서는 다음 조건을 충족해야 합니다.

  • Android 접근성 서비스 활성화
  • 화면 위에 표시(오버레이) 권한 허용
  • 유효한 AI API 키 설정
  • 인터넷 연결

2. 이용자는 본 앱의 이용약관 및 개인정보 처리방침에 동의해야 서비스를 이용할 수 있습니다.

제4조 (이용자의 의무)

1. 이용자는 본 앱을 다음과 같은 목적으로 사용해서는 안 됩니다.

  • 타인의 기기를 무단으로 조작하는 행위
  • 다른 앱이나 서비스의 이용약관을 위반하는 자동화 행위
  • 불법적인 콘텐츠에 접근하거나 유포하는 행위
  • 앱의 보안을 우회하거나 해킹하는 행위
  • 기타 법령에 위반되는 행위

2. 이용자는 자신의 AI API 키를 안전하게 관리할 책임이 있으며, 이로 인한 비용 발생에 대해 운영자는 책임을 지지 않습니다.

3. 이용자는 접근성 서비스를 통해 민감한 정보(비밀번호, 금융 정보 등)가 화면에 표시된 상태에서 에이전트를 실행할 경우, 해당 정보가 AI API로 전송될 수 있음을 인지해야 합니다.

4. 이용자는 세션 코드를 절대 타인에게 공유하거나 공개해서는 안 됩니다. 세션 코드를 알고 있는 제3자는 연결된 기기를 원격으로 조작할 수 있으므로, 세션 코드의 유출로 인해 발생하는 모든 피해에 대해 운영자는 책임을 지지 않습니다.

제5조 (서비스의 제한 및 중단)

1. 본 앱은 다음 경우에 서비스 이용이 제한될 수 있습니다.

  • AI API 서비스 제공자의 장애 또는 정책 변경
  • Android OS 업데이트로 인한 접근성 API 변경
  • 네트워크 연결 불가

2. 운영자는 시스템 유지보수, 기술적 장애, 기타 불가항력적 사유로 서비스를 일시 중단할 수 있습니다.

제6조 (면책 사항)

  1. AI 판단의 정확성: 본 앱의 AI 기반 자동 조작은 AI 모델의 판단에 의존하며, 항상 정확하거나 의도한 대로 동작한다는 것을 보장하지 않습니다.
  2. AI API 비용: 사용자가 설정한 AI API 사용에 따른 비용은 전적으로 사용자의 책임입니다.
  3. 데이터 손실: 자동 조작 과정에서 발생하는 의도치 않은 데이터 변경·삭제에 대해 운영자는 책임을 지지 않습니다.
  4. 제3자 서비스: 외부 AI API 및 Firebase 서비스의 장애·정책 변경으로 인한 서비스 제한에 대해 운영자는 책임을 지지 않습니다.

제7조 (지적재산권)

  1. 본 앱의 소프트웨어, 디자인, 로고 등에 대한 지적재산권은 운영자에게 귀속됩니다.
  2. 이용자는 본 앱을 역컴파일, 리버스 엔지니어링, 또는 소스코드를 추출하여서는 안 됩니다.

제8조 (약관의 변경)

  1. 운영자는 관련 법령에 위배되지 않는 범위에서 이 약관을 변경할 수 있습니다.
  2. 약관이 변경되는 경우, 변경 사항을 앱 내 공지 또는 앱 업데이트를 통해 고지합니다.
  3. 변경된 약관에 동의하지 않는 이용자는 앱 사용을 중단하고 앱을 삭제할 수 있습니다.

제9조 (분쟁 해결)

본 약관에 관한 분쟁은 대한민국 법률을 준거법으로 하며, 분쟁 발생 시 관할 법원은 민사소송법에 따릅니다.

제10조 (시행일)

이 약관은 2026년 2월 26일부터 시행됩니다.
